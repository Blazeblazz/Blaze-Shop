# BLAZE - E-commerce Website

## Overview
BLAZE is an e-commerce website specializing in high-quality t-shirts for the Moroccan market. The website features a modern, responsive design with a focus on user experience and conversion optimization.

## Features
- Responsive design for all devices
- Product catalog with detailed product pages
- Special offers section with pack discounts
- Contact form with backend processing
- Checkout system with order management
- Admin panel for order management
- SEO optimized structure
- Social media integration

## Technical Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP (for backend processing)

## File Structure
- `index.html` - Main homepage
- `product-detail.html` - Product detail page
- `pack-offer.html` - Special offer page for 3-pack
- `checkout.html` - Checkout page
- `css/` - CSS stylesheets
- `js/` - JavaScript files
- `images/` - Image assets
- `admin/` - Admin panel files
- `process-contact.php` - Backend for contact form

## SEO Features
- Meta tags for SEO
- Open Graph tags for social sharing
- Sitemap.xml
- Robots.txt
- Semantic HTML structure

## Server Requirements
- PHP 7.4 or higher
- Apache web server with mod_rewrite enabled
- HTTPS certificate

## Installation
1. Upload all files to your web server
2. Configure the server to use HTTPS
3. Set up proper permissions for the `contacts` directory (777)
4. Update the email in `process-contact.php` to your business email
5. Secure the admin directory with proper authentication
